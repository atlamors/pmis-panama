# PMIS Overview

PMIS is a **Port Management Information System** built with a **micro-frontend** Angular UI and **microservice** .NET backend, connected through an API Gateway.

## Local Defaults

* **Ports (all environments):**
  * Host → `:4200`
  * MFE's → `:4201-4999`
  * API Gateway → `:5000`
  * Auth Service → `:5001`
  * API Services → `:5100-5999`
  * Documentation → `:3000`
* **API Gateway:** `/api/*` (served by backend gateway)
* **Module Federation manifest:** host serves `/assets/mf.manifest.json` as the single source of truth for remote URLs.

---

## What’s Inside

* **Frontend**: Angular 20, Module Federation, Tailwind CSS (v3.4 with v4-style import)
* **Backend**: ASP.NET Core 8, EF Core, YARP gateway
* **Docs**: Nextra + MDX, with Mermaid diagrams

---

## Known Issues

 * **HMR disabled due to infinite reload in dev**: This is a common issue with Module Federation and Angular. It is caused by the way the host app loads the remote app. The host app loads the remote app and then the remote app loads the host app. This causes a loop and the infinite reload.
